分支：相当于文件夹 主分支：项目的原件 功能分支、子分支：项目的复印件 ============================================================= 一、命令 1、查看分支：git branch 2、创建分支：git branch 分支名 3、切换分支：git checkout 分支名 4、创建并切换分支：git branch -b 分支名 注意：切换分支时，确保当前分支工作区是干净的 5、删除分支：git branch -d 分支名 注意：不能在当前分支上自己删除自己； 未被合并的分支，默认不允许删除，但可以强制删除 6、强制删除分支：git branch -D 分支名 7、修改分支名：git branch -m 原分支名 新分支名 8、合并分支：1
